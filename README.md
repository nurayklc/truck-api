# Node.js API 

### Used Techs
- Node.js
- PostgreSQL
- Postman
- MVC Yapısı
- Express Framework
- Nodemon
- Dotenv

---
### Run Project
``` 
npm install
npm start
```
Runs the app in the development mode.
Open http://localhost:3000 to view it in your browser.


---
## Postman Requests

- Parent root -> /vehicle
- Get -> /vehicle_list
- Post-> /vehicle_add
- Patch-> /vehicle_update/:id
- Delete-> /vehicle_delete/:id
